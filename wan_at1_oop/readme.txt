In the hope of reducing server payload and saving time for staff from navigating to different routes to perform data entry, it will be a single page application using web API handling data exchange. 
The project was written in C#. An open source and cross-platform web development framework, ASP.NET Core Blazor, was chosen for the task because it provides a fast, lean and modular environment which fit the needs of the project. 
Entity Framework Core was used to setup SQL database and configure database access.
Data validation for both frontend and backend were achieved by DataAnnotation.
